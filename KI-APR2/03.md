# OOP – datové členy a metody

Objekty (instance tříd) = "chytrá" data

- data: atributy / datové členy
- chování: metody

## Datové členy

Datové členy (atributy) jsou proměnné (data, hodnoty), které jsou součástí objektu. Každý
objekt má své vlastní kopie datových členů. Datové členy mohou být třídní (společné pro
všechny instance třídy) nebo instanční (specifické pro každou instanci třídy).

Třídní datové členy se definují mimo metody třídy a jsou společné pro všechny instance
třídy. Instanční datové členy se definují v konstruktoru třídy a jsou specifické pro každou
instanci třídy.

## Metody

Metody jsou funkce, které jsou součástí objektu. Metody mohou být instanční (operují nad
datovými členy objektu), třídní (nad datovými členy třídy) nebo statické (nepotřebují datové
členy).

## Příklad

Třídní a statické metody se označují dekorátory `@classmethod` a `@staticmethod`.

```python
class Pes:
  krmivo = ['granule', 'maso']    # třídní datový člen

  # konstruktor
  def __init__(self, jmeno: str, majitel: str, zvuk: str):
    # instanční členy
    self.jmeno    = jmeno
    self.majitel  = majitel
    self.zvuk     = zvuk

  # instanční metody mohou číst a měnit instanční i třídní datové členy
  # self: odkaz na objekt
  def stekej(self):
    return f'{self.jmeno}, {self.zvuk}: {self.krmivo}!'

  def zmen_zvuk(self, novy: str):
    self.zvuk = novy

  # třídní metody mohou číst a měnit pouze třídní
  # cls: odkaz na třídu
  @classmethod
  def pridej_krmivo(cls, krmivo):
    cls.krmivo.append(krmivo)

  # statické metody nemají přístup k datům
  @staticmethod
  def jak_dela_pes():
    return "haf haf"
```

```python
azor = Pes('Azor', 'Petr', 'vrrr')
print(azor.stekej())

alik = Pes('Alík', 'Jana', 'ňaf')
print(alik.stekej())

psi = [azor, alik]
for pes in psi: print(pes.stekej())

# volání instančních metod: pomocí objektu
azor.zmen_zvuk('vroom')
alik.zmen_zvuk('fzz')
for pes in psi: print(pes.stekej())

# volání třídních metod: pomocí objektu nebo třídy
azor.pridej_krmivo('knedlík')
for pes in psi: print(pes.stekej())

alik.pridej_krmivo('zelí')
for pes in psi: print(pes.stekej())

Pes.pridej_krmivo('kost')
for pes in psi: print(pes.stekej())

# volání statických metod
print(azor.jak_dela_pes())
print(alik.jak_dela_pes())
print(Pes.jak_dela_pes())
```

## Chráněné a privátní členy

Pokud je potřeba některé datové členy nebo metody skrýt, je možné je označit podtržítky jako
chráněné nebo privátní.

Jména chráněných členů začínají jednoduchým podtržítkem `_`, jména privátních členů dvojitým
podtržítkem `__`.

Chráněné členy jsou přístupné z třídy, v které jsou definované a z podtříd. Privátní členy
jsou přístupné pouze z třídy, v které jsou definované.

# Magické metody

Magické metody jsou speciální metody, které začínají _a končí_ dvojitým podtržítkem. Magické
metody se volají automaticky v různých situacích (např. při vytváření objektu, při porovnání
objektů, při konverzi objektu na řetězec atd.).

Magickým metodám se v Pythonu také říká
[dunder methods](https://en.wikipedia.org/wiki/Dunder) (double underscore).

[Seznam magických metod](https://docs.python.org/3/reference/datamodel.html#special-method-names)

## Konstruktor a destruktor (**finalizer**)

Konstruktor je magická metoda `__init__`, která se volá při vytváření objektu. Destruktor
(finalizátor) je magická metoda `__del__`, která se volá při zrušení objektu.

```python
class Book:
  count: int = 0

  def __init__(self, name: str, abstract: str = ''):
    self.name     = name
    self.abstract = abstract
    print(f'New book: {self.name}')

  def __del__(self):
    print(f'Deleted: {self.name}')
```

```python
lotr = Book('Lord of the Rings', 'One ring to rule them all!')
silm = Book('Silmarillion', 'Ainulindalë, Valaquenta, Quenta Silmarillion, Akallabêth, Of the Rings of Power and the Third Age')
```

```python
lotr = Book('Lord of the Rings')
silm = Book('Silmarillion')
del lotr
```

```python
library: list[Book] = []
library.append(Book('Lord of the Rings'))
library.append(Book('Silmarillion'))
library = []
```

Python používá tzv. reference counting, který počítá, kolikrát je na objekt odkázáno. Teprve
až počet referencí klesne na 0, je objekt zrušen.

```python
lotr = Book('Lord of the Rings', 'One ring to rule them all!')
tolk = lotr
print('deleting lotr')
del lotr
print('deleting tolk')
del tolk
```

### Počítání knih

```python
class Book:
  count: int = 0

  def __init__(self, name: str, abstract: str = ''):
    self.name     = name
    self.abstract = abstract
    print(f'New book: {self.name}')
    Book.count += 1

  def __del__(self):
    Book.count -= 1
    print(f'Deleted: {self.name}')

  @classmethod
  @property     # make the method a "getter", allows calling it without ()
  def get_count(cls) -> int:
    return cls.count

print(Book.get_count)
lotr = Book('Lord of the Rings')
print(Book.get_count)
silm = Book('Silmarillion')
print(Book.get_count)
del silm
print(Book.get_count)
del lotr
print(Book.get_count)
```

## Porovnávání objektů

Magické metody `__eq__`, `__ne__`, `__lt__`, `__le__`, `__gt__`, `__ge__` se volají při
porovnávání objektů.

```python
l1 = Book('Lord of the Rings', 'First edition')
l2 = Book('Lord of the Rings', 'Second edition')

print(l1 == l1) # True, je to stejný objekt (zdědená metoda)
print(l1 != l1) # False

print(l1 == l2) # False, jiný objekt
print(l1 != l2) # True

print(l1 <  l1) # Chyba, nelze porovnat

print(l1 <  l2)
print(l1 <= l2)
print(l1 >  l2)
print(l1 >= l2)
```

```python
class Book:
  def __init__(self, name: str, abstract: str = ''):
    self.name     = name
    self.abstract = abstract

  def __eq__(self, other: Book):
    return self.name == other.name

  def __lt__(self, other: Book):
    return self.name < other.name \
      or self.name == other.name and self.abstract < other.abstract

l1 = Book('Lord of the Rings', 'First edition')
l2 = Book('Lord of the Rings', 'Second edition')

print(l1 == l1) # True, stejné jméno
print(l1 != l1) # False

print(l1 == l2) # True, stejné jméno
print(l1 != l2) # False

print(l1 < l1)  # False
print(l1 < l2)  # True
```

## Řazení objektů

```python

  def __init__(self, name: str, abstract: str = ''):
    self.name     = name
```

Magická metoda `__lt__` je nutná pro řazení objektů.

```python
class Book:
  def __init__(self, name: str, abstract: str = ''):
    self.name     = name
    self.abstract = abstract

  def __lt__(self, other: Book):
    return self.name < other.name \
      or self.name == other.name and self.abstract < other.abstract

l1 = Book('Lord of the Rings', 'First edition')
l2 = Book('Lord of the Rings', 'Second edition')

library = [l1, l2]
```
