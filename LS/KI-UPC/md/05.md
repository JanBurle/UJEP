# Multi-dimensional arrays in C

[Example code](src/12-nd-arrays/arrays.c)

In C, an array can have more than one dimension:

- 1D array: a sequence of values
- 2D array: a table of rows and columns
- 3D array: a stack of tables

Higher-dimensional are fixed-size, contiguous blocks of memory
organized in _row-major_ order: the last index changes the
fastest.

For `int a[3][4]`, memory layout is:
`a[0][0], a[0][1], a[0][2], a[0][3], a[1][0], ...`

Iterating with the last index in the inner loop is usually
faster, due to better cache locality.

```c
for (int i = 0; i < 3; i++) {
	for (int j = 0; j < 4; j++) {
		a[i][j] = i + j;
	}
}
```

## Passing higher-dimensional arrays to functions

When passing a multi-dimensional array, all dimensions except
the first must be known.

```c
void print1D(int arr[], int cols) {
	for (int i = 0; i < cols; i++) {
		printf("%d ", arr[i]);
	}
	printf("\n");
}
```

Equivalent form:

```c
void print1D(int *arr, int cols);
```

```c
void print2D(int arr[][4], int rows, int cols) {
	for (int i = 0; i < rows; i++) {
		for (int j = 0; j < 4; j++) {
			printf("%d ", arr[i][j]);
		}
		printf("\n");
	}
}
```

Equivalent form:

```c
void print2D(int (*arr)[4], int rows, int cols);
```
