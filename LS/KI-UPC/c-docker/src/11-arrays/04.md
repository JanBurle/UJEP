# Pole, ukazatele, řetězce

- pole obsahuje více hodnot stejného typu, indexované od 0,
  uložené souvisle v paměti
- velikost pole je pevně daná při deklaraci, nelze ji měnit
- hranice pole se nekontrolují (riziko přetečení)
- pole se předávají do funkce jako ukazatele, ztrácí se
  informace o velikosti (degradace, decay); velikost se musí
  předat zvlášť (nebo použít sentinel)

- ukazatele jsou proměnné, které obsahují adresu jiné proměnné

- pole a ukazatele jsou úzce spjaty, ale nejsou totéž
- pole alokuje paměť pro data, ukazatel pouze pro adresu

- jméno pole je konstantní ukazatel na první prvek
- pole nelze přiřadit celé najednou, ukazatel lze změnit, kam
  ukazuje

- řetězce jsou speciální typ: pole znaků ukončených nulovým
  znakem (bytem), `0` nebo `'\0'`

## Jednorozměrná pole

### Deklarace pole

- `typ název[velikost];`
- velikost musí být konstantní výraz
- prvky jsou indexované od 0 do velikost-1

`array_declaration()`

### Inicializace pole

`array_initialization()`

### Přístup k prvkům

- indexový operátor `[]`, index od 0 do velikost-1
- dereference ukazatele (degradace pole na ukazatel)

`array_access()`

- `*(pole + i)` je ekvivalentní `pole[i]`
- `pole[i]` je syntaktické cukrátko pro `*(pole + i)`
- funguje i naopak: `i[pole]` je totéž jako `pole[i]`

`array_access2()`

### Aritmetika s ukazateli

- `p++` posune ukazatel na další prvek
- `p--` posune ukazatel na předchozí prvek
- `p + n` posune ukazatel o n prvků dopředu
- `p - n` posune ukazatel o n prvků zpět
- `p1 - p2` vrací počet prvků mezi ukazateli

- lze porovnávat ukazatele na prvky stejného pole, operátory:
  `==`, `!=`, `<`, `>`, `<=`, `>=`

`pointer_arithmetic()`

Průchod polem pomocí ukazatele, ukazatel za konec pole

`pointer_arithmetic2()`

### Ukazatel na pole vs. pole ukazatelů

`array_pointer()`

### Výměna hodnot (swap)

- Klasický příklad předávání odkazem
- Příklad:
  ```c
  void prohod(int *a, int *b) {
      int temp = *a;
      *a = *b;
      *b = temp;
  }
  int x = 10, y = 20;
  prohod(&x, &y);  // x=20, y=10
  ```

### Vracení více hodnot

- Funkce může vrátit jen jednu hodnotu
- Pomocí ukazatelů lze "vrátit" více hodnot
- Příklad:
  ```c
  void vydel_se_zbytkem(int delenec, int delitel,
                        int *podil, int *zbytek) {
      *podil = delenec / delitel;
      *zbytek = delenec % delitel;
  }
  int p, z;
  vydel_se_zbytkem(17, 5, &p, &z);  // p=3, z=2
  ```

### Pole jako argumenty

- Pole se vždy předává jako ukazatel
- Funkce může měnit prvky pole
- Velikost se musí předat zvlášť
- Příklad:
  ```c
  void vynuluj(int *pole, int velikost) {
      for (int i = 0; i < velikost; i++) {
          pole[i] = 0;
      }
  }
  ```

### Konstantní ukazatele

- **Ukazatel na konstantu**: `const int *p` - nelze měnit
  hodnotu
- **Konstantní ukazatel**: `int * const p` - nelze změnit
  adresu
- **Obojí**: `const int * const p`
- Příklad:
  ```c
  void tiskni(const int *pole, int velikost) {
      // pole[0] = 10;  // CHYBA - nelze měnit
      printf("%d", pole[0]);  // OK - lze číst
  }
  ```

### Výhody předávání ukazatelem

- Umožňuje změnu původní hodnoty
- Efektivní pro velké datové struktury (nepředává se kopie)
- Umožňuje vracení více hodnot
- Nutné pro práci s poli

## Práce s řetězci jako s poli znaků

### Reprezentace řetězců

- Řetězec je pole znaků ukončené nulovým znakem `'\0'`
- Typ `char[]` nebo `char*`
- Nulový znak označuje konec (není součást viditelného textu)
- Příklad:
  ```c
  char text[6] = {'H', 'e', 'l', 'l', 'o', '\0'};
  // nebo jednodušeji:
  char text[] = "Hello";  // automaticky přidá '\0'
  ```

### Deklarace a inicializace řetězců

- **Pole znaků**:
  ```c
  char str[20];           // neinicializované
  char str[20] = "Hello"; // inicializované, zbývající znaky '\0'
  char str[] = "Hello";   // velikost 6 (včetně '\0')
  ```
- **Ukazatel na řetězec**:
  ```c
  char *str = "Hello";    // řetězcová literál (read-only)
  ```

### Rozdíl mezi polem a ukazatelem

- **Pole** `char str[] = "Hello";`
  - Kopie v paměti (lze měnit)
  - Alokováno na zásobníku
- **Ukazatel** `char *str = "Hello";`
  - Ukazuje na literál (nelze měnit)
  - Literál v read-only paměti
  - Příklad:

    ```c
    char pole[] = "Hello";
    pole[0] = 'h';  // OK

    char *ukazatel = "Hello";
    ukazatel[0] = 'h';  // CHYBA - nedefinované chování
    ```

### Čtení a výpis řetězců

- **Výpis**: `printf("%s", str);` nebo `puts(str);`
- **Čtení**:
  ```c
  char str[50];
  scanf("%s", str);        // bez & (jméno pole je adresa)
  fgets(str, 50, stdin);   // bezpečnější, načte i mezery
  ```

### Procházení řetězce

- **Pomocí indexu**:
  ```c
  char str[] = "Hello";
  for (int i = 0; str[i] != '\0'; i++) {
      printf("%c", str[i]);
  }
  ```
- **Pomocí ukazatele**:
  ```c
  char *p;
  for (p = str; *p != '\0'; p++) {
      printf("%c", *p);
  }
  ```

### Kopírování řetězců

- **Nelze** použít `=` pro kopírování
- Musí se kopírovat znak po znaku
- Příklad:

  ```c
  char zdroj[] = "Hello";
  char cil[10];
  // cil = zdroj;  // CHYBA

  // Ruční kopírování:
  int i;
  for (i = 0; zdroj[i] != '\0'; i++) {
      cil[i] = zdroj[i];
  }
  cil[i] = '\0';
  ```

### Velikost vs. délka řetězce

- **sizeof**: velikost pole v bytech (včetně '\0' a
  nepoužitých míst)
- **strlen**: počet znaků před '\0'
- Příklad:
  ```c
  char str[20] = "Hello";
  sizeof(str);   // 20 (celé pole)
  strlen(str);   // 5 (délka "Hello")
  ```

### Pole řetězců

- Dvourozměrné pole nebo pole ukazatelů
- Příklad:
  ```c
  char jmena[3][20] = {"Alice", "Bob", "Charlie"};
  // nebo
  char *jmena[] = {"Alice", "Bob", "Charlie"};
  ```

## Standardní funkce pro práci s řetězci

### Knihovna string.h

- Obsahuje funkce pro manipulaci s řetězci
- Nutné includovat: `#include <string.h>`
- Většina funkcí pracuje s null-terminated strings

### strlen() - Délka řetězce

- **Syntaxe**: `size_t strlen(const char *s)`
- Vrací počet znaků před '\0' (nepočítá '\0')
- Příklad:
  ```c
  char str[] = "Hello";
  int len = strlen(str);  // len = 5
  ```

### strcpy() - Kopírování řetězce

- **Syntaxe**: `char *strcpy(char *dest, const char *src)`
- Kopíruje src do dest (včetně '\0')
- Nekontruje velikost cíle - nebezpečné!
- Příklad:
  ```c
  char zdroj[] = "Hello";
  char cil[10];
  strcpy(cil, zdroj);  // cil = "Hello"
  ```

### strncpy() - Bezpečné kopírování

- **Syntaxe**:
  `char *strncpy(char *dest, const char *src, size_t n)`
- Kopíruje maximálně n znaků
- Pokud src < n, doplní nulami
- Nemusí přidat '\0' na konec!
- Příklad:
  ```c
  char cil[10];
  strncpy(cil, "Hello", sizeof(cil) - 1);
  cil[sizeof(cil) - 1] = '\0';  // zajistit ukončení
  ```

### strcat() - Spojení řetězců

- **Syntaxe**: `char *strcat(char *dest, const char *src)`
- Připojí src na konec dest
- Nekontruje velikost - nebezpečné!
- Příklad:
  ```c
  char str[20] = "Hello ";
  strcat(str, "World");  // str = "Hello World"
  ```

### strncat() - Bezpečné spojení

- **Syntaxe**:
  `char *strncat(char *dest, const char *src, size_t n)`
- Připojí maximálně n znaků ze src
- Vždy přidá '\0' na konec
- Příklad:
  ```c
  char str[20] = "Hello ";
  strncat(str, "World", sizeof(str) - strlen(str) - 1);
  ```

### strcmp() - Porovnání řetězců

- **Syntaxe**: `int strcmp(const char *s1, const char *s2)`
- Vrací:
  - 0 pokud s1 == s2
  - < 0 pokud s1 < s2
  - > 0 pokud s1 > s2
- Porovnává lexikograficky (podle ASCII)
- Příklad:
  ```c
  if (strcmp(str1, str2) == 0) {
      printf("Řetězce jsou stejné\n");
  }
  ```

### strncmp() - Porovnání n znaků

- **Syntaxe**:
  `int strncmp(const char *s1, const char *s2, size_t n)`
- Porovnává prvních n znaků
- Příklad:
  ```c
  if (strncmp("Hello", "Help", 3) == 0) {
      printf("První 3 znaky jsou stejné\n");
  }
  ```

### strchr() - Hledání znaku

- **Syntaxe**: `char *strchr(const char *s, int c)`
- Vrací ukazatel na první výskyt znaku c
- Vrací NULL, pokud nebyl nalezen
- Příklad:
  ```c
  char str[] = "Hello";
  char *p = strchr(str, 'l');  // p ukazuje na první 'l'
  if (p != NULL) {
      printf("Nalezen na pozici %ld\n", p - str);
  }
  ```

### strrchr() - Hledání znaku od konce

- **Syntaxe**: `char *strrchr(const char *s, int c)`
- Vrací ukazatel na poslední výskyt znaku c
- Příklad:
  ```c
  char str[] = "Hello";
  char *p = strrchr(str, 'l');  // p ukazuje na druhé 'l'
  ```

### strstr() - Hledání podřetězce

- **Syntaxe**:
  `char *strstr(const char *haystack, const char *needle)`
- Vrací ukazatel na první výskyt needle v haystack
- Vrací NULL, pokud nebyl nalezen
- Příklad:
  ```c
  char text[] = "Hello World";
  char *p = strstr(text, "World");
  if (p != NULL) {
      printf("Nalezeno: %s\n", p);  // "World"
  }
  ```

### strtok() - Tokenizace

- **Syntaxe**: `char *strtok(char *str, const char *delim)`
- Rozdělí řetězec na tokeny podle oddělovačů
- Modifikuje původní řetězec!
- První volání s řetězcem, další s NULL
- Příklad:
  ```c
  char str[] = "Jmeno,Prijmeni,Vek";
  char *token = strtok(str, ",");
  while (token != NULL) {
      printf("%s\n", token);
      token = strtok(NULL, ",");
  }
  ```

### sprintf() - Formátování do řetězce

- **Syntaxe**:
  `int sprintf(char *str, const char *format, ...)`
- Jako printf, ale výstup do řetězce
- Příklad:
  ```c
  char buffer[50];
  int x = 42;
  sprintf(buffer, "Hodnota je %d", x);
  ```

### sscanf() - Čtení z řetězce

- **Syntaxe**:
  `int sscanf(const char *str, const char *format, ...)`
- Jako scanf, ale čte z řetězce
- Příklad:
  ```c
  char str[] = "42 3.14";
  int x;
  double y;
  sscanf(str, "%d %lf", &x, &y);
  ```
