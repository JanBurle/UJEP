# Python ve VSCode

Jak použít VSCode pro programování v Pythonu?

- [Python ve VSCode](https://code.visualstudio.com/docs/languages/python)
- [Python/VScode - Debugging](https://code.visualstudio.com/docs/python/debugging)

## Použití lokální instalace Pythonu

### Nastavení VSCode a projektu

- zkopírujte si [šablonu projektu](../vscode-python/)
- otevřete složku ve VSCode
- zkontrolujte, že máte
  [doporučená rozšíření](../vscode-python/.vscode/extensions.json)
  pro Python:
  - Python (`ms-python.python`)
  - Python Debugger (`ms-python.debugpy`)
  - Pylance (`ms-python.vscode-pylance`)
  - Ruff (`charliermarsh.ruff`)
- ostatní nastavení můžete vypnout (disable)

### Práce v terminálu

- oevřete soubor [test.py](../vscode-python/test.py)
- otevřete vestavěný terminál
- navigujte do složky projektu
- spusťte Python (`python test.py` nebo `python3 test.py`)
- soubor změňte a znovu spusťte, atd...
- soubor můžete automaicky formátovat pomocí Ruff

### Práce s debuggerem

- zkontrolujte
  [konfiguraci spouštění](../vscode-python/.vscode/launch.json)
- otevřete soubor [test.py](../vscode-python/test.py) a
  spusťte ho pomocí `F5` nebo zelené šipky v levém horním rohu
  záložky "Run and Debug"
- nastavte breakpointy pomocí `F9` nebo kliknutím do levého
  okraje editoru

# Úlohy ze zápočtového testu

## Úloha 1

Vytvořte funkci random_choice, která přijímá seznam řetězců a
vrací (pseudo)náhodně jakýkoliv z nich. Přitom se však vyhýbá
případným prázdným řetězcům (tj. pokud je v seznamu prázdný
řetězec, nikdy ho nevrátí). Pokud je seznam prázdný nebo
obsahuje jen prázdné řetězce (a tudíž není, co vrátit), měla
by být vyvolána výjimka ValueError.

```python
random_choice(["", "Frodo", "Sam", "", "Pipin"])
# vrací "Frodo" resp. "Sam" resp. "Pipin"
# (pseudonáhodně s rovnoměrným rozložením)
```

## Úloha 2

Vytvořte funkci similar_lists, která přijímá dva číselné
seznamy a, b (s položkami třídy float resp. int) a vrátí True,
pokud jsou přibližně shodné. Pokud tato podmínka neplatí, je vrácen objekt False. Pokud
mají oba seznamy různý počet prvků, resp. oba jsou prázdné,
pak je vyvolána výjimka `ValueError`.

```python
similar_lists([1, -2.0], [1.00024,
-1.99976])
# vrací True
```

## Úloha 3

Vytvořte program, který zapíše soubor `six_seven.txt` obsahující
všechna čísla od 1 do 1992, pro něž platí, že jsou dělitelná 6
a zároveň je součet jejich cifer dělitelný 7. Čísla by měla
seskupena po deseti na jednom řádku (oddělená mezerou).
Poslední řádek může být kratší.

Sumu cifer (číslic), pokud možno realizujte jako funkci.

```python
six_seven.txt:
498 588 678 696 768 786 858 876 894 948 966 984
1398 1488 1578 1596 1668 1686 1758 1776 1794 1848 1866 1884
1938 1956 1974 1992
```

## Úloha 4

Vytvořte funkci `group_by`, která přijímá
seznam řetězců a vrací slovník, jenž mapuje celá čísla (klíče)
na řetězce (hodnoty). Slovník seskupuje řetězce podle délky
řetězce, tj. všechny řetězce vstupního seznamu se stejným
počtem znaků jsou spojeny do jednoho velkého řetězce (s
oddělovačem `", "`), přičemž klíčem tohoto seznamu je tato
společná délka.

```python
group_by(["Frodo", "Bilbo", "Boromir", "Faramir"])
# výsledek: {5: "Bilbo, Frodo", 7: "Boromir, Faramir"}
```
