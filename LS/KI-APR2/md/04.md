# ADT – abstraktní datové typy, sekvenční datové struktury

- **datová struktura**: způsob organizace dat v paměti (datový
  typ)
- **ADT**, _abstraktní_ datový typ: koncepční model, který
  definuje sadu operací a chování pro datovou strukturu
  (interface, rozhraní), _aniž by specifikoval_, jak jsou tyto
  operace implementovány, ani jak jsou data organizována v
  paměti.
- **abstrakce** ("odtažení"): skrývání detailů

- ADT: https://www.geeksforgeeks.org/abstract-data-types/
- ADT:
  https://medium.com/@tssovi/abstract-data-type-adt-in-python-33e6ce1f961e

### Příklady ADT

Některé ADT jsou v Pythonu vestavěné, jiné je třeba importovat
z knihoven nebo implementovat.

- seznam (`list`)
- seřazený seznam (ordered list)
- ntice (`tuple`)
- množina (`set`)
- mapa/asociativní pole (`dict`, dictionary)
- fronta (queue)
- prioritní fronta (priority queue)
- fronta s dvěmi konci (double-ended queue, deque)
- zásobník (stack)
- spojový seznam (linked list)
- dvojitý spojový seznam (double-linked list)
- graf (graph)
- strom (tree)

## Fronta (queue)

ADT typu FIFO (first-in, first-out). Prvky se přidávají vzadu
(back, rear), odebírají zepředu (front).

Definice (např.):

- `Queue() → queue`: konstruktor prázdné fronty
- `isEmpty() → bool`: je fronta prázdná? (tzv. predikát)
- `size() → int`: vrací počet prvků ve frontě
- `push(item) → None`: přidá prvek dozadu do fronty, jako
  poslední
- `front() → item`: vrátí první prvek z fronty (zepředu)
- `pop() → item`: vrátí _a odebere_ první prvek z fronty

Implementace:

```python
class Queue:
  def __init__(self):
    self._items = []

  def isEmpty(self):
    return 0 == self.size()

  def size(self):
    return len(self._items)

  def push(self, item): # also: put / enqueue
    self._items.append(item)

  def front(self):
    # raises an exception if empty
    return self._items[0]

  def pop(self):        # also: get / dequeue
    # raises an exception if empty
    return self._items.pop(0)

  def __str__(self):
    return str(self._items)

q = Queue()
q.push("Alena")
q.push(True)
q.push(0)
q.push("Pavel")
q.push(5.6)

print(q.pop())
print(q.pop())
print(q)
print(q.front())
print(q.pop())
print(q.pop())
print(q.pop())
# print(q.pop()) # IndexError: pop from empty list
```

S vlastní výjimkou:

```python
class QueueError(Exception):
  def __init__(self, msg = 'Empty queue'):
    super().__init__(msg)

class Queue:
  def __init__(self):
    self._items = []

  def isEmpty(self) -> bool:
    return 0 == self.size()

  def size(self) -> int:
    return len(self._items)

  def __assert_not_empty(self):
    if self.isEmpty():
      raise QueueError()

  def push(self, item):
    self._items.append(item)

  def front(self) -> object:
    # guard, raises an exception if empty
    self.__assert_not_empty()
    return self._items[0]

  def pop(self) -> object:
    # guard, raises an exception if empty
    self.__assert_not_empty()
    return self._items.pop(0)

  def __str__(self):
    return str(self._items)

q = Queue()
print(q.pop()) # QueueError
```

Nebo bez výjimky, s návratovou hodnotou `None`:

```python
class Queue:
  def __init__(self):
    self._items = []

  def isEmpty(self) -> bool:
    return 0 == self.size()

  def size(self) -> int:
    return len(self._items)

  def push(self, item):
    self._items.append(item)

  def front(self) -> object|None:
    return None if self.isEmpty() else self._items[0]

  def pop(self) -> object|None:
    return None if self.isEmpty() else self._items.pop(0)

  def __str__(self):
    return str(self._items)

q = Queue()
print(q.pop()) # None
```

`Queue` a `PriorityQueue`:

## Zásobník (stack)

ADT typu LIFO (last-in, first-out). Prvky se přidávají i
odebírají na jednom konci.

Definice (např.):

- `Stack() → stack`: konstruktor
- `isEmpty() → bool`: je prázdný?
- `size() → int`: kolik má prvků
- `push(item)`: přidá prvek na zásobník
- `top() → item`: vrátí prvek z vrcholu zásobníku
- `pop() → item`: vrátí _a odebere_ prvek z vrcholu zásobníku

Python má třídu `LifoQueue`, kterou lze použít jako zásobník:

```python
from queue import LifoQueue
help(LifoQueue)
```

## ❖ Úloha

Implementujte zásobník (Stack), podobně jako Queue: s vlastní
výjimkou `StackError` nebo s návratovou hodnotou `None`, a s
volitelným parametrem `maxSize`.

## Příklad použití zásobníku

Páry závorek: zkontrolujeme, zda jsou v textu správně vyváženy
(uzávorkovány).

```python
from queue import LifoQueue as Stack

def checkBraces(s: str):
  '''Checks if pairs of braces () [] {} are balanced.'''
  stack = Stack()
  for c in s:
    if c in '([{':
      c = ')]}'['([{'.index(c)] # what the heck?
      stack.put(c)
    elif c in ')]}':
      if c != stack.get():
        print('unbalanced:', c)
        return

  if not stack.empty():
    print('Unbalanced!')
  else:
    print('Balanced!')

checkBraces('([{}])')
checkBraces('([{])}')
```

## ❖ Úloha

- Ve funkci `checkBraces()` použijte místo `LifoQueue` váš
  `Stack`.
- Přidejte další páry znaků, které by měly být vyváženy: `<>`
  a `/\`.
- Pokud narazíte na nevyvážený pár závorek, vypište i pozici v
  textu, na které se chyba vyskytla.
- Při chybě vypište text a označte chybný znak, např. takto:
  ```
  Chyba: ([{])}
            ^ očekáváno: }
  ```

Implementujte a použijte/otestujte dvojitou frontu

Definice (např.):

- `Dequeue() → dequeue`: konstruktor prázdné fronty
- `isEmpty() → bool`: je fronta prázdná? (tzv. predikát)
- `size() → int`: vrací počet prvků ve frontě
- `push_back(item) → None`: přidá prvek dozadu do fronty -
  `push_front(item) → None`: přidá prvek dopředu do fronty
- `front() → item`: vrátí první prvek z fronty
- `back() → item`: vrátí poslední prvek z fronty
- `pop_front() → item`: vrátí _a odebere_ první prvek z fronty
- `pop_back() → item`: vrátí _a odebere_ první prvek z fronty

## ❖ Úloha: seřazený seznam

Implementujte a použijte/otestujte seřazený seznam (ordered
list).

Definice (např.):

- `OrderedList() → list`: konstruktor
- `isEmpty() → bool`
- `size() → int`
- `add(item) → None`: přidá prvek
- `get(index) → item`: vrátí prvek na dané pozici
- `pop(index) → item`: odebere a vrátí prvek na dané pozici

Možné strategie řazení:

1. Po přidání prvku (append) vždy seřaďte (sort) seznam.
2. Před přidáním prvku najděte (sekvenčně) správnou pozici, na
   kterou prvek vložit (insert).
3. Podobně jako 2., ale pozici prvku hledejte půlením
   intervalu (binary search).
4. Prvek přidávejte na konec jako v případě 1., seznam označte
   jako neseřazený ("dirty"), a teprve až při vyžádání prvku
   (get, pop) seřaďte seznam, pokud nutno.

## ❖ Implementujte seřazený seznam

### Strategie 1, řazení po každém přidání prvku:

```python
class OrderedListError(Exception):
  def __init__(self, msg: str):
    super().__init__(msg)

class OrderedList:
  def __init__(self):
    self._items = []

  def isEmpty(self) -> bool:
    return not self.size()

  def size(self) -> int:
    return len(self._items)

  def add(self, item):
    self._items.append(item)
    self._items.sort()
    return self # for chaining of add().add().add()...

  def get(self, i: int) -> object:
    try:
      return self._items[i]
    except:
      raise OrderedListError('bad index')

  def pop(self, i: int) -> object:
    try:
      return self._items.pop(i)
    except:
      raise OrderedListError('bad index')

  def __str__(self):
    return str(self._items)

# test

o = OrderedList()
o.add(2).add(1).add(4).add(3)
print(o)
for i in range(o.size()): print(i, o.get(i))
```

### Strategie 2, řazení při vkládání:

Prototyp kódu, který sekvenčně (O(n)) nalezne pozici pro
vložení prvku `x` a přidání prvek na tuto pozici:

```python
lst = [1,2,3,3,4]

def findPos(x: int):
  for i,val in enumerate(lst):
    if x < val: return i
  return len(lst)

def add(x: int):
  i = findPos(x)
  lst.insert(i, x)

add(2.2)
print(lst)
```

### Strategie 3, řazení při vkládání, _binární vyhledávání_:

Rekurzí:

```python
lst = [1,2,3,4,5]

def _findPos(x, lo, hi):
  if lo == hi: return lo
  mid = (lo + hi) // 2
  return _findPos(x,lo,mid) if x<lst[mid] else _findPos(x,mid+1,hi)

def findPos(x):
  return _findPos(x, 0, len(lst))
```

Iterací:

```python
def findPos(x):
  lo = 0
  hi = len(lst)

  while lo < hi:
    mid = (lo + hi) // 2
    if x < lst[mid]:
      hi = mid
    else:
      lo = mid+1
  return lo
```

### Strategie 4, řazení až při vyžádání prvku:

```python
class OrderedList:
  def __init__(self):
    self._items = []
    self._dirty = False # dirty = needs sorting

  def add(self, item):
    self._items.append(item)
    self._dirty = True
    return self # for chaining

  def __make_clean(self):
    if self._dirty:
      self._items.sort()
      self._dirty = False

  def get(self, i: int) -> object:
    self.__make_clean()
    try:
      return self._items[i]
    except:
      raise OrderedListError('bad index')

  def pop(self, i: int) -> object:
    self.__make_clean()
    try:
      return self._items.pop(i)
    except:
      raise OrderedListError('bad index')
```
