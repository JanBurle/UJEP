### ❖ Úloha: Vývojové prostředí

Podle návodu si nainstalujte a nastavte vývojové prostředí:
Docker, VSCode, Chrome.

### ❖ Úloha: Inspekce serveru

Spusťte virtuální server. Otevřete si interaktivní shell (v
produkčním prostředí – skutečný server v oblacích – byste
použili `ssh`):

```bash
docker exec -ti ... bash

# v kontejneru si prohlédněte:
php -v                   # ověřte verzi PHP
php -i | less            # konfigurace PHP
php -i | grep php.ini    # konfigurace PHP
php -i | grep xdebug     # debugger?
php -i | grep short      # short_open_tag?

mc
cd /usr/local/etc/php    # konfigurace PHP
cd /etc/apache2          # konfigurace Apache
cd /var/www              # website root, mapped from outside OR NOT?

# ^D ukončí shell
```
