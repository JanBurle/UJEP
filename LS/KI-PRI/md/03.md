# Responsivní design a reaktivní UI

- responsivní stránka: adaptuje se na různé rozměry (okénka,
  zařízení)
- reaktivní stránka: reaguje na změny dat a stavu

## [RWD: responsive web design](https://www.w3schools.com/css/css_rwd_intro.asp)

Tři základní části RWD:

- viewport `<meta>` značka
- flexibilní rozvržení stránek
- media queries

### Viewport `<meta>` značka

Bez
[této značky](https://www.w3schools.com/css/css_rwd_viewport.asp)
mobilní prohlížeč zobrazí (?) stránku jako desktopovou (šířka
980px) a následně ji zmenší, aby se vešla na obrazovku. Na
desktopu nemá tato značka žádný účinek.

```html
<meta
  name="viewport"
  content="width=device-width, initial-scale=1.0"
/>
```

Hodnoty v atributu `content`:

- `width=`
  - `device-width`: nastaví šířku viewportu na šířku zařízení
  - konkrétní hodnota v px (např. 600)
- `initial-scale=`
  - `1.0`: nastaví počáteční zoom (přiblížení) na 100%
  - jiné hodnoty (např. 0.5 pro 50% zoom)
- `user-scalable=`
  - `yes`: (default) povolí přibližování (zoom) uživatelem
  - `no`: zakáže přibližování (zoom) uživatelem

Další hodnoty:

- `maximum-scale`
- `minimum-scale`
- `height` (méně běžné)

### Media queries

["at-rules"](https://www.w3schools.com/cssref/atrule_media.php)

```css
@media [<type> and] (<condition>) {
    ... rules
}
```

- type: `all` (default), `print`, `screen`, `speech`
- condition: feature(s), and or not, ... (comma = or)
- feature: `min-width: ??px`, `max-width: ??px`, `width`,
  `orientation: portrait/landscape`,
  `prefers-color-scheme: light/dark`, ...

```css
@media (min-width: 300px) and (max-width: 700px) { ... }
@media (300px <= width <= 700px) { ... }
@media screen and (min-width: 768px) and (prefers-color-scheme: dark) { ... }
```
