# Úvod do HTML a CSS

## HTML (HyperText Markup Language)

### 1991

HTML vznikl jako koncept, který obsahoval základní značky pro jednoduché formátování textu a odkazy. Bez formální specifikace.

### 1995

HTML 2.0, první oficiální standardizovaná verze. Zavedla základní strukturu dokumentu (`<html>`, `<head>`, `<body>`) a také formuláře (`<form>`): první krok k interaktivitě.

- [RFC 1866](https://www.rfc-editor.org/rfc/rfc1866) ("Request for Comments")

### 1997

HTML 3.2, první verze schválená konsorciem W3C. Zavedla tabulky, applety, obtékání textu kolem obrázků.

Probíhaly války mezi prohlížeči: Netscape a IE přidávaly vlastní nestandardní značky.

- [HTML 3.2 Specification](https://www.w3.org/TR/REC-html32)

### 1999

HTML 4.01, na dlouhou dobu (~15 let) stabilní verze. Tři typy dokumentů (Strict, Transitional, Frameset). Důraz na oddělení obsahu od prezentace (použití CSS) a podporu JavaScriptu.

- [HTML 4.01 Specification](https://www.w3.org/TR/html401/)

### 2000

XHTML 1.0. Pokus o striktní HTML pomocí syntaxe XML (všechny tagy musí být ukončené, psány malými písmeny atd.). Příliš se neujalo.

- [XHTML 1.0 Specification](https://www.w3.org/TR/xhtml1/)

### 2014 - stále:

HTML5. Veliká změna. Nejen značkovací jazyk, ale _plnohodnotná platforma pro webové aplikace_.

- sémantické značky (`<header>`, `<nav>`, `<article>`, `<section>`, `<aside>`, `<footer>`)
- vestavěná ("nativní") podpora multimédií (`<video>`, `<audio>`)
- grafické elementy (`<svg>`,`<canvas>`)
- lokální úložiště (localStorage), webové komponenty, WebSockets, Service Workers ... a mnoho jiného

Verze se dále nečíslují, HTML5 se vyvíjí se jako [živý standard](https://html.spec.whatwg.org) pod správou skupiny WHATWG (Web Hypertext Application Technology Working Group - Apple, Mozzila, Opera, Google).

Výrobci prohlížečů implementují nové funkce postupně:

Souhrný status:

- https://caniuse.com/
- https://developer.mozilla.org/
- https://html.spec.whatwg.org/
- https://web-platform-tests.org/

Jednotliví výrobci:

## CSS (Cascading Style Sheets)

### 1996

CSS 1: První oficiální doporučení W3C. Cíl: Oddělit vzhled od obsahu (HTML). Základní formátování: fonty, barvy, pozadí, zarovnání textu a okraje (margin, padding, border).

- [CSS 1 Specification](https://www.w3.org/TR/REC-CSS1/)

### 1998

CSS 2: Rozvržení stránek (layout), pozicování prvků (absolute, relative, fixed), z-index, typy médií (pro obrazovky, pro tisk), pseudo-třídy (např. `:hover`).

Ne zcela úspěšné, jen částečně implementováno prohlížeči, vznikaly další požadavky (responzivní design, animace, flexibilní rozvržení).

- [CSS 2 Specification](https://www.w3.org/TR/1998/REC-CSS2-19980512/)

### 2011

CSS 2.1: Revize 2.0. De facto standardem na mnoho let, stále neměla moderní funkce.

- [CSS 2.1 Specification](https://www.w3.org/TR/CSS21/)

### 1999 – stále:

CSS 3 (od 1999 – dodnes):

Velké změny. Rozděleno na monoho nezávislých modulů, dovoluje nezávislý vývoj.

Pokročilé řízení vzhledu, průhlednost a stíny,animace, přechody. Pružná rozložení (Flexbox, Grid), CSS proměnné a funkce, vnořené selektory.
