# Ukázkový projekt

[zde](../full-project2/)

Mimo jiné:

- Použití PHP pro generování HTML obsahu, zpracování dat a
  logiku na straně serveru
- Použití `require()` pro modularizaci a znovupoužitelnost
  kódu
- Připojení k databázi pomocí PDO (PHP Data Objects)
- Prevence SQL injection, připravené příkazy
- PHP sessions a cookies pro správu stavu
- Zpracování formulářů a validace dat
