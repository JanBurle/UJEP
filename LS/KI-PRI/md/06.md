# HTTP protokol a REST API

- HTTP protokol: cyklus požadavek/odpověď, metody (GET, POST, ...)
- Stavové kódy: 2xx, 3xx, 4xx, 5xx, význam a použití
- Hlavičky: Content-Type, Authorization, Cache-Control
- Principy REST API: jednotné rozhraní, bezstavová komunikace
- Lokální vývojové prostředí (Docker, Docker Compose)
