# Datový formát JSON

[JSON](https://www.w3schools.com/js/js_json.asp) (JavaScript
Object Notation - syntax pro zápis objektů v JavaScriptu) je
jednoduchý datový formát pro ukládání a přenos dat.

Má jen několik málo základních datových typů:

### Jednoduché typy:

| Typ                       | Příklad          |
| ------------------------- | ---------------- |
| text (string)             | `"Hello"`        |
| číslo (number)            | `42`             |
| logická hodnota (boolean) | `true` / `false` |
| prázdná hodnota (empty)   | `null`           |

Text je uzavřen ve dvojitých uvozovkách, ostatní jsou bez
uvozovek.

### Složené typy:

| Typ                      | Příklad              |
| ------------------------ | -------------------- |
| seznam/pole (list/array) | `[1, 2, 3]`          |
| objekt                   | `{ "name": "John" }` |

Pole je uspořádaná, indexovatelná kolekce hodnot, které mohou
být jakéhokoliv datového typu (včetně polí a objektů).

Objekt je kolekce párů klíč-hodnota, kde klíč je textový
řetězec a hodnota může být jakýkoliv datový typ (včetně polí a
objektů).

[Syntaxe](https://www.w3schools.com/js/js_json_syntax.asp) a
[datové typy](https://www.w3schools.com/js/js_json_datatypes.asp).

Příklad JSON objektu:

```json
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "hobbies": ["reading", "traveling"],
  "address": {
    "street": "123 Main St",
    "city": "Anytown"
  },
  "partner": null
}
```
