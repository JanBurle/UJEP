# Responsivní design

Ve své HTML stránce:

- použijte meta viewport značku
  - ověřte její funkci v emulátoru cílového zařízení
  - vyzkoušejte různé hodnoty atributu `content` a sledujte,
    jak se změní chování na mobilu vs. desktopu

- vytvořte pružný (fluidní) layout pomocí procent, flexboxu a
  gridu, například:
  - jedno/dvou/třísloupcové rozložení, které se příslušně mění
    / přizpůsobuje šířce okna
  - galerii karet, která se přizpůsobuje šířce okna
    - pomocí flexboxu i gridu, porovnejte výhody/nevýhody

- pružné rozvržení vytvořte oběma strategiemi: mobile-first a
  desktop-first, a porovnejte je z hlediska čitelnosti a
  údržby

- použijte media query pro orientaci portrait/landscape
