# Začínáme s HTML5 a CSS3

## Struktura HTML dokumentu (webové stránky)

### Kostra dokumentu

Každá HTML5 stránka má tuto základní strukturu:

```html
<!DOCTYPE html>
<html lang="cs">
  <head>
    <meta charset="UTF-8" />
    <meta
      name="viewport"
      content="width=device-width, initial-scale=1.0"
    />
    <title>Titulek stránky</title>
    <link rel="stylesheet" href="style.css" />
  </head>
  <body>
    <!-- Zde viditelný obsah stránky -->
  </body>
</html>
```

- `<!DOCTYPE html>`: HTML5, žádné číslo verze jako dříve. Toto
  _není značka_.
- `<html lang="cs">`: hlavní obal (kořenový prvek), může
  definovat jazyk stránky
- `<head>`: neviditelná hlavička, obsahuje metadata
- `<meta>`: různé informace o dokumentu
- `<link>`: odkaz na externí zdroj
- `<body>`: tělo, viditelné v záložce (okně) prohlížeče.

### Sémantické značky

Místo *nad/zne*užívání obecného `<div>` používá HTML5 popisné
značky.

Příklad typického rozložení těla sttránky:

```html
<body>
  <header>
    <!-- záhlaví stránky -->
    <nav>
      <!-- navigace -->
    </nav>
  </header>

  <main>
    <!-- hlavní blok obsahu stránky -->
    <h1>Nadpis článku</h1>

    <article>
      <!-- smysluplný celek (např. článek, příspěvek na fóru) -->
      <p>Text článku...</p>
    </article>

    <section>
      <!-- logická sekce obsahu -->
    </section>
  </main>

  <aside>
    <!-- postranní pane -->
  </aside>

  <footer>
    <!-- patička -->
  </footer>
</body>
```

## CSS styly: oddělení formy (vzhledu) od obsahu (HTML struktury)

- HTML říká: _toto je nadpis_
- CSS říká: _nadpis/y bude/ou modrý/é a takto velký/é_

### Selektory a vlastnosti

CSS styl je sekvence pravidel. CSS pravidlo se skládá ze
selektoru (na co se pravidlo vztahuje) a bloku deklarací
(vlastností a hodnot).

```css
selektor {
  vlastnost: hodnota;
  vlastnost: hodnota;
  ...
}
```

Příklad stylu nadpisu:

```css
h1 {
  color: darkblue;
  font-size: 24px;
}
```

Hlavní typy selektorů:

| Typ           | Příklad   | Co vybere                               | Příklad                              |
| ------------- | --------- | --------------------------------------- | ------------------------------------ |
| značka (tag)  | `p`       | všechny odstavce `<p>`                  | `p { color: grey; }`                 |
| třída (class) | `.btn`    | prvky s `class="btn"`                   | `.btn { background: red; }`          |
| ID            | `#logo`   | prvek s `id="logo"`                     | `#logo { width: 100px; }`            |
| univerzální   | `*`       | všechny prvky                           | `* { box-sizing: border-box; }`      |
| potomek       | `nav a`   | odkazy `<a>`, které jsou uvnitř `<nav>` | `nav a { text-decoration: none; }`   |
| přímý potomek | `nav > a` | `<a>`, které jsou přímo v `<nav>`       | `nav > a { text-decoration: none; }` |
| pseudo-třída  | `:hover`  | prvek při najetí myší                   | `a:hover { color: orange; }`         |

### Kaskáda stylů (Cascade)

Působnost pravidel pro stejnou vlastnost se "kaskádovitě"
kombinuje. Kdo vyhraje, o tom rozhoduje váha pravidla:

- důležitost `!important`: přebije vše (nezneužívat)
- specificita: "inline" styl (přímo ve značce `style="..."`)
  **>** ID **>** třída **>** značka **>** univerzální selektor
  - příklad: `#logo` převáží `.btn`, to převáží `p`
- pořadí v kódu: pokud mají pravidla stejnou váhu, platí
  pravidlo, které je uvedeno později

### Příklad vlastností (atributů)

- `color`, `font-family`, `font-size`, `text-align` - práce s
  textem
- `width`, `height`, `padding`, `margin`, `border` - box model
- `background-color`, `background-image` - pozadí
- `border-width` (tloušťka), `border-style` (typ čáry: solid,
  dashed, none), `border-color` (barva) - rámeček
- `display` - chování boxu (block, inline, flex, grid)

## Box model

Rozložení prvků (layout) na stránce určuje CSS Box Model
(krabicový model). Každý prvek na stránce (nadpis, obrázek,
odstavec) má obdelníkový tvar (box).

Box má 4 slupky (vrstvy), směrem od středu ven:

- _content_ (obsah)
- _padding_ (vnitřní okraj): prostor mezi obsahem a rámečkem
- _border_ (rámeček): hranice, viditelná (čára s danou
  tloušťkou) nebo neviditelná
- _margin_ (vnější okraj): průhledný prostor kolem rámečku
  (bez pozadí)

### Počítání velikosti (`box-sizing`)

Výchozí chování (`box-sizing: content-box`): `width` a
`height` určují pouze velikost obsahu. Padding, border a
margin se přičítají k celkové velikosti prvku.

Moderní řešení (`box-sizing: border-box`): `width` a `height`
určují celkovou velikost prvku včetně rámečku a okrajů.
Většinou používáme tento mód, protože je logičtější. Proto se
v CSS na začátku často píše tento "reset":

```css
* {
  box-sizing: border-box;
}
```

## Validace HTML a CSS

Prohlížeče se snaží být tolerantní k chybám v HTML a CSS a
mnohé odpouštějí. Přesto je vhodné kód validovat.

- Oficiální W3C Validátory (Online)
  - [validator.w3.org](https://validator.w3.org)
  - [jigsaw.w3.org/css-validator](https://jigsaw.w3.org/css-validator/)
- Validace přímo v IDE (VSCode)
- Vývojářské nástroje (console) v prohlížeči
