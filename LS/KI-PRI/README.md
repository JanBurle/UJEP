# Programování pro internet

## Informace o předmětu

Kurs poskytuje studentům přehled základních technologií pro
vývoj webových aplikací se zaměřením na získání praktických
dovedností. Studenti si osvojí principy tvorby obsahu stránek
pomocí HTML5 a CSS3, včetně responsivity a reaktivity. Získají
praktickou zkušenost s programováním v jazycích JavaScript a
PHP, s fungováním HTTP protokolu a REST API, s datovými
formáty JSON a XML, a s funkcí webového serveru včetně
napojení na databázi. Diskutovány budou také základní
optimalizace pro vyhledávače (SEO) a přehled moderních
vývojových frameworků.

## Podmínky získání zápočtu, zkouška

Podmínkou získání zápočtu je úspěšné vypracování seminární
práce, která spočívá ve vytvoření webového projektu, v němž
budou použity probírané technologie. Student při zápočtu
předvede svůj webový projekt (fungující 😉) a je schopný jej
obhájit. Mimo jiné musí student znát svůj kód, být schopný
vysvětlit jeho funkci a případně na místě provést malé změny.

Zkouška bude mít písemnou a ústní část, ve které student
prokáže znalost probíraných technologií a principů webového
vývoje.
