# Základní vlnové funkce

[Sinusová vlna](https://en.wikipedia.org/wiki/Sine_wave):

> y(t) = A·sin(2π·f·t + φ)

- y(t) : okamžitá hodnota v čase t [s]
- A : amplituda [1]
- f : frekvence [Hz]
- počáteční fáze [rad]

Zjednodušeně:

> y(t) = A·sin(ω·t + φ)

- ω = 2π·f : úhlová frekvence (angular frequency) [rad/s]

#### setup

```python
import numpy as np
sr  = 48000 # good sampling rate
pi  = np.pi # π
pi2 = 2*pi  # 2π

from IPython.display import Audio
import matplotlib.pyplot as plt
```

#### sinusovka

```python
dur = 2    # duration, seconds
frq = 440  # frequency (Hz, tone A4)
amp = .5   # amplitude

# time points
t = np.linspace(0, dur, int(sr*dur), endpoint=False)

# computed sine wave
wave = amp * np.sin(pi2 * frq * t)

print(len(t))
print(min(wave), max(wave))
print(t[0:10])
```

#### přehrát

```python
Audio(wave, rate=sr)
```

(cvaknutí na konci?)

#### zobrazit

```python
plt.figure(figsize=(8, 3))
plt.plot(t, wave)
plt.title(f"Sine wave {frq} Hz")
plt.xlabel("Time (seconds)")
plt.ylabel("Amplitude")
plt.grid(True)
plt.show()
```

moc husté

#### zobrazit začátek

```python
n = 200
plt.figure(figsize=(8, 3))

```

```python
n = 200
```

#### zapsat do souboru
