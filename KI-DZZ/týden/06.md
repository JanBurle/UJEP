# Fyzikální/fyzické modelování zvuku

## Karplus-Strongův algoritmus pro syntézu zvuku struny

S využitím našeho [toolboxu](./02a.md) + šumy:

```python
class KarplusStrong(Wave):
  def __init__(self, dur: Dur, amp: Amp, frq: Frq, decay: float = 0.996, fade=True):
    self.decay = decay
    super().__init__(dur, amp, frq, 0, fade)

  def _make(self, dur: Dur, amp: Amp, frq: Frq, phs: Phs, fade: bool):
    n_smp = toSmp(dur)
    N = int(sr / frq)  # delay line length
    line = np.random.uniform(-1, 1, N).astype(Flt) # excitation / white noise
    out = np.zeros(n_smp, Flt) # output buffer

    for i in range(n_smp): # loop
      out[i] = line[i % N]
      avg = self.decay * (line[i % N] + line[(i+1) % N])/2 # simple LP filter
      line[i % N] = avg

    out = out / np.max(np.abs(out))   # normalize to [-1, 1]
    self.buf = (amp * out).astype(Flt)

    if fade:
      self._fade()
```

```python
audio = Audio()

for i,frq in enumerate([220, 440, 660, 880]):
  wave = KarplusStrong(1.6, .8, frq, fade=False)
  audio.add(wave, i)

audio.plot(.1)
audio.show()
```

# Csound (1986, MIT)

Doménově-specifický jazyk pro syntézu zvuku a programování hudby.

Modulární a rozšiřitelný systém založený jednotkových generátorech (unit generators).

Dva textové soubory: orchestr `.orc` a notový zápis `.sco`. Oba lze spojit do jednoho `.csd`.

https://csound.com

v prohlížeči

https://ide.csound.com/profile/JanBurle

Prefixy:

- g = global
- i = init (once)
- k = control (slow)
- a = audio (fast)
- s = string/text
- p = score parameter
