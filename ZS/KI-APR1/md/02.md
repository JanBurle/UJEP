## ƒå√≠sla typu `float`

Desetinn√° (re√°ln√°) ƒç√≠sla t≈ô√≠dy (typu) `float` jsou ‚Äûnep≈ôesn√°‚Äú, to jest: jsou p≈ôesn√°, ale jen pro urƒçit√© hodnoty, kter√© jsou realizovateln√© v hardware. V√Ωsledky v√Ωpoƒçt≈Ø s nimi se mohou zaokrouhlit na nejbli≈æ≈°√≠ reprezentovateln√© ƒç√≠slo.

- [IEEE-754 for geeks](https://www.geeksforgeeks.org/computer-organization-architecture/ieee-standard-754-floating-point-numbers/)
- [Python: Floating-Point Arithmetic](https://docs.python.org/3/tutorial/floatingpoint.html)
- [Calculator](https://www.h-schmidt.net/FloatConverter/IEEE754.html) (zkuste nap≈ô. `0.1`)

Nepou≈æ√≠vejme je proto pro p≈ôesn√© v√Ωpoƒçty a p≈ôesn√° porovn√°v√°n√≠!

P≈ô√≠klad:

```python
5/7 == (2/7) * (5/2) # should be True, but it is not
```

> False

Proƒç?

```python
5/7
```

> 0.714285714285714**3**

```python
(2/7) * (5/2)
```

> 0.714285714285714**2**

Rozd√≠l:

```python

```

> 1.1102230246251565e-16 # velmi mal√© ƒç√≠slo (numerick√°, zaokrouhlovac√≠ chyba)

Zde bylo ƒç√≠slo vyps√°no v tzv. _vƒõdeck√©m z√°pisu_ (scientific notation), ve form√°tu _mantissa_ a _exponent_.

V√Ωhodou ƒç√≠sel `float` je jejich velk√Ω rozsah, nev√Ωhodou jejich omezen√° p≈ôesnost.

`float` ƒç√≠sla lze _p≈ôibli≈ænƒõ_ porovnat nap≈ô. takto (`abs` je nutn√©!!):

```python
abs((5/7) - ((2/7) * (5/2))) < 1e-10
```

To jest, pt√°me se: je rozd√≠l ƒç√≠sel dostateƒçnƒõ mal√Ω, neboli: jsou si ƒç√≠sla dostateƒçnƒõ bl√≠zk√°?

Modul `math` m√° pro takov√©to porovn√°n√≠ funkci `isclose()`:

```python
math.isclose(5/7, 2/7 * 5/2)
```

## ≈ò√°dkov√Ω v√Ωstup

Pro ≈ô√°dkov√Ω v√Ωstup je k dispozici _vestavƒõn√°_ funkce `print()`: [w3schools](https://www.w3schools.com/python/ref_func_print.asp), [docs.python](https://docs.python.org/3/library/functions.html#print)

`print()` um√≠ vypsat hodnoty mnoha (v≈°ech?) typ≈Ø:

```python
print(3)
print(3.2)
print(3/2)
print('3/2')
```

Lze tak√© vypsat nƒõkolik hodnot najednou. Jednotliv√© hodnoty jsou zad√°ny jako tzv. _pozicion√°ln√≠ argumenty_ (positional arguments), a jsou automaticky oddƒõleny mezerou:

```python

```

Oddƒõlovaƒç (obsah mezi vypsan√Ωmi hodnotami, separator) lze specifikovat pomoc√≠ tzv. _pojmenovan√©ho_ argumentu (named argument) `sep`:

```python

```

`print` autovaticky ukonƒç√≠ ≈ô√°dku t√≠m, ≈æe na jej√≠ konec p≈ôid√° (neviditeln√Ω) znak pro p≈ôechod na novou ≈ô√°dku (_newline_, _NL_).

```python
print('ƒç√≠sla: ')
print(3, 3.2, 3/2, '3/2')
```

Ukonƒçen√≠ ≈ô√°dky lze specifikovat pomoc√≠ pojmenovan√©ho argumentu `end`:

```python
print('ƒç√≠sla: ', end='') # no newline
print(3, 3.2, 3/2,'3/2')
print('ƒç√≠sla: ', end='// ') # custom ending
print(3, 3.2, 3/2,'3/2')
```

A v takov√©m p≈ô√≠padƒõ lze _NL_, p≈ôechod na novou ≈ô√°dku vlo≈æit pomoc√≠ tzv. ‚Äûescape‚Äú sekvence `\n`:

```python

print(3, 3.2, 3/2, '3/2')
```

## Porovn√°v√°n√≠ a podm√≠nƒõn√Ω v√Ωraz (tern√°rn√≠ oper√°tor)

## Booleovsk√° algebra (boolean algebra)

V Booleovsk√© algeb≈ôe jsou

- dvƒõ hodnoty: `False` a `True` (_ne_ a _ano_, 0 a 1)
- t≈ôi z√°kladn√≠ oper√°tory:
  - `and`, logick√Ω souƒçin (konjunkce)
  - `or`, logick√Ω souƒçet (disjunkce)
  - `not`, negace

```python
print(type(False), type(True), 3<4, type(3<4)) # etc.
print(3<4 and 4<5)
print(3<4 and 3>4)
print(3<4 or 3>4)
print(not 3>4)
```
