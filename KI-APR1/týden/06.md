Pohyb a zatáčení:

```python
T.initializeTurtle()
T.forward(100)
T.right(90)
T.forward(100)
T.back(200)
T.left(90)
T.forward(100)
```

Tvar, rychlost, domů, otáčení:

```python
T.initializeTurtle()
T.shape('circle')
T.speed(7)
for deg in range(0,360,60):
  T.home()
  T.setheading(deg)
  T.forward(60)
```

Šířka čáry a další:

```python
T.initializeTurtle(8, (400,300))
T.shape('circle')
T.color('red')
T.bgcolor('yellow')
for i in range(6):
  T.width(i+1)
  T.pendown()
  T.forward(10)
  T.penup()
  T.forward(10)
```

Přerušovaná čára:

```python
T.initializeTurtle()
T.shape('circle')

centerX = T.window_width()  // 2
centerY = T.window_height() // 2
for i in range(6):
  T.penup()
  T.goto(centerX + i*10, centerY + i*5)
  T.pendown()
  T.forward((i+1) * 20)
```

Barvy:

```python
T.initializeTurtle()
T.shape('circle')

T.color('red')
T.forward(20)
T.color('green')
T.forward(20)
```

Strom:

```python
T.initializeTurtle(8, (400,300))
T.shape('circle')
T.width(1)

T.penup()
T.goto(T.window_width() // 2, T.window_height())
T.pendown()

def tree(len):
  if len<10: return

  T.fd(len)

  T.lt(20)
  tree(len * .6)
  T.rt(40)
  tree(len * .6)
  T.left(20)

  T.bk(len)

tree(80)
```
